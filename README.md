# Moux World

A dark-themed social role-play platform featuring a digital passport system, global world feed, real-time chat, and an integrated digital economy.

## Features

- **Digital Passports & Profiles**: Customizable user profiles with avatars, status ranks, login streaks, and personalized badges.
- **Real-Time Communication**: Engage in the global World Chat or connect via secure private Direct Messaging.
- **Social Feed**: Share updates, media, and interact with the community feed.
- **Community Servers**: Create, manage, and join dedicated community servers with administrative controls.
- **Digital Economy**: Integrated Moux Balance system, marketplace, and inventory management for digital assets and crypto.
- **Account Security & Moderation**: Robust user management including Two-Factor Authentication (2FA), blocking, muting, and automated moderation systems.

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Backend / Database**: Firebase (Firestore, Authentication)
